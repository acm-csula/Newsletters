Our Projects!

Lowerclassmen
Game Development


ACM's Beginners Game Development is a fun and exciting 6-week project. We designed and created this project to help those students seeking to head into the game development field and for those trying to gain game development experience

During the 6-week project, students will be working in groups of two. Students will create a Space Invaders copy and will learn the needed Python and Python module, Pygame, commonly used to create 2D retro arcade games. Students will also learn about sprites, sprite movement, collision detection, uploading images, sounds, level changing, player lives, and the logic behind the scenes making it all come together. This project will be fast-paced and challenging but is designed with beginners skill level in mind


Upperclassmen
Web Development


Participate in creating a chat web application similar to Discord with the use of Node.js, React.js, JavaScript, JSX, and Firebase

Everyone will contribute to one project and create a Single-Page chat web application. The goal is to teach you the basics of web development with React. You will learn to Conceptualize and Design a web application that utilizes industry used technology. You will also learn common practices and code management that will help you become a better web and applications developer

If you are looking for a challenge and want to learn new technologies that are not common in CSULA's CS program, then this is the project to be a part of. Web development experience is expected, but it is not a requirement
